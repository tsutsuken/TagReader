#  TagReader

## 概要

NFCタグの読み取りをするアプリです。  
CoreNFCフレームワークを利用しています。

## 使用技術

- Xcode 14.2
- Swift 5
- SwiftUI
- CoreNFC
  - NFCNDEFReaderSession
  
## 機能一覧

- NFCタグの情報を読取る
- 読み取ったNFCタグの情報を表示する
  - テキスト
  - URL
